# Textbook: Sorting Strings by Length (KATA_SORT_04)

## Theoretical Foundation
When creating context tokens, formatting dynamic tables, or optimizing prompt spaces, sorting text segments by character lengths is highly pragmatic.

---

## Syntax Anatomy
```python
words = ["cat", "elephant"]
shortest_first = sorted(words, key=len) # ["cat", "elephant"]
```
