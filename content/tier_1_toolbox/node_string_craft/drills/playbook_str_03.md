# Textbook: Replacing Substrings (KATA_STR_03)

## Theoretical Foundation
Anonymizing sensitive information (like API keys, passwords, or personal names) before caching prompts or logging outputs is a critical security rule. The `.replace()` method provides a performant, character-exact search-and-replace mechanism across text documents.

---

## Step-by-Step Exercise Instructions
1. Call `text.replace("Leonidas", "Justin")` on the input string.
2. Return the sanitized string.
