## Theoretical Foundation
In computer programming, a variable is the most fundamental building block. Think of a variable as a labeled storage box inside your computer's memory. To store data, we write the variable name (the label), an equals sign (=), and the value we want to store.

Example:
```python
score = 100
```
This tells Python to reserve a space in memory, label it "score", and store the number 100 inside it.

### Anatomy of a Function Definition (def) and return
A function is a reusable block of code that performs a specific action.
*   **def:** Short for "define". It tells Python we are starting to write a function.
*   **get_player_score():** The name of the function, followed by parentheses which can hold inputs.
*   **Colon (:):** Signals the start of the block of code belonging to the function.
*   **return:** Tells Python to hand back the calculated value to whoever called the function.

---

## Syntax Anatomy
```python
def get_player_score():
    score = 100
    return score
```
