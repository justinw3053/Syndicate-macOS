# Textbook: Collecting Items - Lists (KATA_GRAM_04)

## Theoretical Foundation
Sometimes we want to store multiple items together. A `list` is an ordered collection of elements grouped inside brackets `[` and `]`, separated by commas.

Lists can hold numbers, strings, or even other lists.

Example:
```python
tokens = ["hello", "world"]
```

---

## Syntax Anatomy
```python
def get_simple_list():
    tokens = ["hello", "world"]
    return tokens
```
