## Theoretical Foundation
Computers are world-class calculators. In Python, we can perform basic math operations using standard operators:
*   `+` (Addition)
*   `-` (Subtraction)
*   `*` (Multiplication)
*   `/` (Division)

When we pass variables into functions (called parameters), we can perform math on them directly.

Example:
```python
total = a + b
```

---

## Syntax Anatomy
```python
def add_numbers(a, b):
    total = a + b
    return total
```
