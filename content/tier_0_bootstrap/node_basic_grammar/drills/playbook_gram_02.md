## Theoretical Foundation
In computer science, raw text is called a "string" because it represents a chain (or string) of individual characters. To write a string in Python, we must always wrap the text inside quotation marks (either single quotes `'` or double quotes `"`). 

If we forget the quotes, Python tries to interpret the text as variable names or commands, leading to syntax crashes!

Example:
```python
username = "Justin"
```

---

## Syntax Anatomy
```python
def get_username():
    username = "Justin"
    return username
```
