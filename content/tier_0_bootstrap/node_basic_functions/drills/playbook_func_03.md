# Textbook: Adding to Lists - Append (KATA_FUNC_03)

## Theoretical Foundation
Lists are dynamic arrays. We add new items to the end of any list using the `.append(item)` method.

---

## Step-by-Step Exercise Instructions
1. Append the string `"EOF"` directly to the `tokens` list using `tokens.append("EOF")`.
2. Return the modified `tokens` list.
