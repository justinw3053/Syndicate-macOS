# Textbook: Accumulating Totals in Loops (KATA_FLOW_04)

## Theoretical Foundation
To sum up list values inside a loop, we initialize an accumulator variable set to `0.0` before the loop starts. Inside the loop, we use the addition operator `+=` to add each number to our running total.

---

## Step-by-Step Exercise Instructions
1. Initialize an accumulator `total = 0.0`.
2. Loop through `numbers` using a `for` loop.
3. Add each `num` to `total` using `total += num`.
4. Return `total` at the end of the function.
