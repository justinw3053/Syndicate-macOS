## Theoretical Foundation
In Python, there is a strict difference between:
*   A single equals sign `=` (used to **assign** a value to a variable).
*   A double equals sign `==` (used to **check** if two values are equal).

Example:
```python
status = "active"       # Assigns "active" to status
if status == "active":  # Checks if status is currently equal to "active"
    return True
```
