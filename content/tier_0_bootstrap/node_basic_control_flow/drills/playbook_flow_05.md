# Textbook: Filtering Items in Loops (KATA_FLOW_05)

## Theoretical Foundation
Filtering is the pattern where we loop through a list and append only items that match specific criteria to a new list container. We initialize an empty list `filtered = []` before starting the loop.

---

## Step-by-Step Exercise Instructions
1. Initialize an empty list: `filtered = []`.
2. Loop through `numbers` using a `for` loop.
3. Inside the loop, check if the current number is strictly positive (`num > 0`).
4. If it is, call `filtered.append(num)` to add it.
5. Return the `filtered` list after the loop completes.
